About

MyDoctor is a complete solution for searching doctors according to their location and category. Doctors can register themselves dynamically by small registration fee by paypal or credit card and that fee can be setup from admin panel by admin. If doctors want to show themselves in the featured list then they need to pay more by paypal or credit card. This CMS Includes almost everything you need to find the right doctor. MyDoctor is fully responsive for any device with tow attractive templates. Admin can manage complete website without having a single line of coding knowledge. It has strong SQL injection protection system which will keep away this system from hackers. This script creates an unique session for each login every time and checks login session is valid with this system or not. So there is no question about session hijacking.

Installation

After downloading file from codecanoyn just extract in your computer and you will get three folders:

Step 1 : Extract files.zip from Files folder to server.

Step 2 : Create a new database in your server.

Step 3 : Now check your domain(Website Name), you will get an installation page.

Step 4 : Enter your installation details then click on Install.
To get purchase code, follow this instruction or contact us: Get Purchase Code

Step 5 : Please wait until the installation is complete. After completing installation, it will show you a success message.

Step 6 : Then click on Close & Run Website (You must click it).

Admin

This is admin profile section. Admin can change account details from here. So admin doesn't need to do this manually from database. In order to do that, Admin will have click on the under arrow beside the profile photo. From there admin will get three different options Edit Profile, Change Password and Logout. From Edit Profile admin can change his account details. From Reset Password Admin can also Update his password. And if admin wants to logout from this panel, he or she can logout from here by clicking Logout option.

Dashboard

"MyDoctor" dashboard is very simple, informative and dynamic. Admin can see details about total Doctors, Categories & Ads. from the Dashboard. Admin can also see the chart of Total Referrals and Most Used OS.

Doctors

Admin can create new doctor by clicking on the Add Doctor from the top right corner of Admin panel. To add new doctor admin will have to fillup all the fields from Add New Doctor page.

Admin can change the details of an added doctor from Edit Doctor page.

Admin can also Active/Deactive a doctor and Remove an added doctor from Doctors page.

Categories

Admin can manage categories from Categories section. If wants to add new category admin will have to click on Add New Category from the top right corner of Categories page & fill the all fields.

Slug must be in English and should be small letters because it will be used as Website URL.

Admin can also Update and Remove added category.

Advertisements

"MyDoctor" offers two types of Advertisement support. You can add advertisement by script or you can add advertisement by banner and advertisement URL.

Admin can add advertisement to click on Add New Advertisement from the top right corner of Admin panel. To add new advertisement admin have to select Advertisement type then complete the simple steps from Add New Advertisement page.

Admin can Edit, Deactivate or Remove advertisements by single click and Deactivated advertisements can be Activated by single click too.

Testimonials

Admin can manage testimonials from Testimonials section. If wants to add new testimonial admin will have to click on Add New Testimonial from the top right corner of Testimonial page & fill the all fields.

Admin can also Edit & Remove added Testimonial.

Blog Section

Admin can manage blogs from this section. To add a new blog admin will have to click on Add New Blog from the top right corner of Admin panel.To add new blog admin have to fill the form from Add New Blog page.

Admin can Edit and Remove a blog by single click.

Google Analytics

"MyDoctor" offers you easily add Google Analytics integration system in your site, it is a freemium web analytics service offered by Google that tracks and reports website traffic. Admin just need to put Google Analytics script in the form and everything will be done.

Meta Keywords

Admin can add meta keywords in website by writing them in this form. Every keyword must be separated using comma. It will help admin for better SEO.
